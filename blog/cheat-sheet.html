Enumerables

Enumerables give you many useful ways of doing something to every element of a collection object (an array or hash, for instance), which is a very common thing to do when you're building programs and websites.

An enumerable must provide a method #each in its class to yield successive members of the collection. #each calls the given block once for each element in self, passing that element as a parameter. For example: 

> [1,2,3].each { |num| print "#{num}! " }
1! 2! 3! => [1,2,3]
Two of the most commonly used enumerables are #map and #select.

Enumerables: #map and #select

#map Takes an array.
my_array = [2, 4, 6, 8, 100]
